what am I looking at?

This page is a complete, working GPT. Not a connection to one — the transformer itself: … parameters of fp16 weights baked into this single HTML file, executed in vanilla JavaScript, no libraries, no network, no GPU. It was trained from scratch with a NumPy-only implementation (hand-derived backpropagation, gradient-checked against finite differences) on the public-domain tiny-shakespeare corpus.

Output — the model writes one character at a time. The amber glow on the text is the attention distribution of the newest character: the brighter a character glows, the more the model attended to it just now. Watch it lock onto a speaker's name when it writes dialogue.

Next character — before each character is placed, the model produces a probability for every symbol it knows. The bars show the top candidates after temperature and top-k; the amber bar is the one the dice actually picked. Low temperature = obedient, high temperature = adventurous.

Attention — the same signal as the glow, but separated per head and layer. The engine uses an incremental KV-cache, so each character costs ~0.7M multiply-adds; when the 128-character context fills, the window slides and the cache rebuilds (the ⟲ flash).

Honesty note — a ~0.6M-parameter character model is a bonsai tree, not a forest: it writes Shakespeare-shaped text, inventing words with period-correct spelling and stage direction. That's the point: small enough to see through, real enough to be the same machinery as the big ones.
